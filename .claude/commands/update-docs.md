## Update Docs

PR の変更内容を分析し、関連ドキュメントの更新・作成を行う。

### 手順

1. **変更内容の把握**
   - デフォルトブランチとの差分から変更されたファイル・機能を特定

2. **ドキュメント影響の判定**
   - 以下のドキュメントに更新が必要か確認:
     - `README.md`: 使い方、セットアップ手順、機能説明
     - `CLAUDE.md`: コマンド、アーキテクチャ、ワークフロー
     - `docs/` 配下: 設計ドキュメント、ガイド

3. **更新・作成の実行**
   - 既存ドキュメントは該当箇所のみ更新（他の部分は変更しない）
   - 新機能やディレクトリ追加時は必要に応じて新規ドキュメントを作成

4. **差分の確認**
   - 更新内容のサマリーを表示

### 制約

- PR の変更に関連する部分のみ更新（無関係な箇所は触らない）
- 日本語と半角英数字の間に半角スペースを入れる
